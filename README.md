### Projects

#### [Sales Insights Dashboard](https://github.com/Monisa-Analyst/sales-insights-dashboard)
A sales dashboard built with Streamlit and SQLite. I wanted to practice building a clean data pipeline and database schema from scratch using real-world e-commerce sales data.
- Normalized a raw dataset of 50k sales records into a clean 5-table star schema.
- Wrote the ETL scripts in Python using Pandas to clean up messy date formats, currency symbols, and missing values.
- Integrated a conversational interface using the Gemini API so you can ask questions like "who is our top customer in 2015?" and it converts the question to SQL to query the database. If the API limit runs out, it falls back to a custom query parser I wrote.
- Included analytical SQL scripts showing window functions, CTEs, and joins.

#### [AI SOC Analyst](https://github.com/Monisa-Analyst/ai-soc-analyst)
A Python tool to help security analysts triage logs and summarize incidents.
- Wrote scripts to parse security log formats and identify suspicious patterns.
- Used LLM prompts to summarize long, noisy logs into quick incident context so analysts can make decisions faster.
